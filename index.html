What I'm Learning

Semantic HTML

Semantic HTML uses meaningful tags like <header>, <main>, and <footer> to structure content clearly. This helps both users and search engines understand the layout of a page.

Accessibility

Accessibility means making websites usable for everyone, including people with disabilities. I use alt text for images, proper labels for forms, and ARIA attributes where needed.

SEO Basics

SEO (Search Engine Optimization) helps websites appear in search results. I include descriptive titles, meta tags, and relevant keywords in my content.
